function startBatch()

Begins a batch operation. Basically it refrains from constructing the layout grid, which causes a reflow, on each item added. Instead, The grid is only generated at the end once wcLayoutTable.finishBatch is called.

Type Definitions

typedef var tableItem

The table item is an object that represents one cell in the layout table, it contains convenient methods for cell alteration and supports chaining. Its purpose is to remove the need to alter <tr> and <td> elements of the table directly.

Properties:
Name Type Description
$ jQuery~Object If you truely need the table cell jQuery object, here it is.
css module:wcLayoutTable~tableItem_css Wrapper to alter jQuery's css function.
stretch module:wcLayoutTable~tableItem_stretch More reliable method for setting the table item width/height values.
Version: 3.0.0
